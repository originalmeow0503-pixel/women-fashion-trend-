Project Method

What this project does

  • I cleaned the Kaggle dataset and parsed `p_attributes` into usable fashion features.
  • I derived `age_group`, `occasion`, `trend_score`, and `trend_label` using rules and scoring logic.
  • I trained a KNN classifier offline and exported the model outputs as static JSON files.
  • This website reads those exports and presents them as a shopping-style frontend.

Limitations

What this demo does not claim

  • `trend_label` is an engineered proxy target, not a real customer ground-truth label.
  • `age_group` and some `occasion` values are inferred from product text and attributes.
  • The website is a static demo of exported notebook results, not a live production system.
  • Popularity is approximated using product ratings and rating count because transaction data is not available.
