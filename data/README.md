# Data Folder

This project already uses:

- `data/raw/Fashion Dataset.csv`

The notebook reads the raw CSV from `data/raw/` and writes the cleaned machine learning dataset to:

- `data/processed/fashion_trend_dataset.csv`

Main outputs created from the raw Kaggle dataset:
- cleaned product-level table
- engineered age groups
- engineered occasion labels
- engineered style categories
- trend score and trend label for KNN
- frontend-ready JSON files for the website
